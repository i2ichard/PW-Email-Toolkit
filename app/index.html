Sup.

Here's a toolkit you can use for the countless emails you have to code..

Most browsers and email clients work well with modern HTML5 and CSS3 code but we have to take into account the people that still use old platforms like IE9 and older as well as older versions of outlook.

To play nicely on all platforms, we're going oldschool and using nested tables to lay out our emails.


To start things off, here is a general layout and wrapper for all emails:


                        
<div style="background-color: #ffffff; height: 100%; margin-top: 0;
         margin-right: 0; margin-bottom: 0; margin-left: 0; padding-top: 0; padding-right: 0;
         padding-bottom: 0; padding-left: 0; width: 100%;">
    <!-- FULL WRAP -->
    <table align="center" border="0" cellpadding="0" cellspacing="0"
             style="height: 100%; -webkit-font-smoothing: antialiased; -webkit-text-size-adjust:
             none; -ms-text-size-adjust: none; width: 100%;">
        <tr>
            <td align="center" bgcolor="#ffffff" valign="top">
                ALL EMAIL CODE GOES IN HERE
                Get ready to start nesting tables..
            </td>
        </tr>
    </table>
</div>

Every email starts off with these.

Pre-header
